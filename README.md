## About the Project

**YENİ-EL** is a research project funded by TÜSEB (Türkiye Sağlık Enstitüleri Başkanlığı) under the *B Group Project Support Program* (Call **2025-B-01**, Project No. **48276**). It develops a multi-DOF, AI-supported bionic upper-limb prosthesis for below-elbow amputees.

The system fuses **surface EMG (sEMG)** and **Force Myography (FMG)** signals, runs adaptive ML models on microcontrollers (TinyML / TensorFlow Lite), and evaluates user performance with clinical instruments (ACMC, SHAP, CRT) and a **pose-based visual assessment protocol** using MediaPipe / OpenPose.

## Tech Stack

- **MCUs:** STM32, ESP32, Nordic nRF52
- **ML/DL:** PyTorch, TensorFlow Lite — CNN, LSTM, Transformer, SNN
- **Pose estimation:** MediaPipe, OpenPose
- **CAD / EDA:** SolidWorks / FreeCAD, KiCad
- **Languages:** Python, C/C++
